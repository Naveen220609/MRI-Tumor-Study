# MRI Brain Tumor Classification and Analysis

## Overview

This project focuses on brain tumor classification using deep learning techniques applied to MRI scans. It includes a comparative study of convolutional neural networks and transformer-based architectures, along with evaluation metrics and explainable AI methods.

---

 ## Results

Generated outputs include:

* Learning curves
* ROC curves
* Confusion matrices
* Model comparison plots

---

## Explainability (XAI)

The project includes interpretability techniques such as:

* Grad-CAM, provide insights into model decision-making in a medical imaging context.
